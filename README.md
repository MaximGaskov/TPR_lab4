# TPR_lab4
### Подготовка к запуску
Установите:
* [python 3.7](https://www.python.org/downloads/)
* pandas
* xlrd
* openpyxl

Необходимые пакеты можно установить с помощью `pip`:
```bash
pip install pandas
pip install xlrd
pip install openpyxl
```

### Запуск программы
Запуск через терминал.  
Входные данные вводятся в таблицу `input.xlsx`

Пример запуска:
```bash
python lab4.py
```
